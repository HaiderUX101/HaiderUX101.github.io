Introduction to HTML

HTML stands for Hyper Text Markup Language and is the backbone of most websites. It provides the structure of a webpage by using a series of elements and tags.

Basic Structure of an HTML Document

Every HTML document begins with the <!DOCTYPE html> declaration, which tells the browser to render the page using HTML5. The rest of the document is divided into two main parts:

  • Head Section: Contains meta-information, the title, links to CSS files, scripts, etc.
  • Body Section: Contains the actual content of the webpage that users see.

Common HTML Elements

  • <html>: The root element that encloses the entire HTML document.
  • <head>: Contains metadata and links to resources.
  • <title>: Specifies the title of the document, which appears in the browser tab.
  • <body>: Contains all the content that is displayed to the user.
  • <h1> to <h6>: Define headings, with <h1> being the most important.
  • <p>: Defines a paragraph of text.
  • <a>: Creates hyperlinks to other pages or resources.
  • <img>: Embeds images into the page. Remember to use the alt attribute to provide alternative text.

Example of a Simple HTML Page

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
    <p>This is a simple paragraph to demonstrate basic HTML elements.</p>
    <a href="https://www.example.com">Visit Example</a>
</body>
</html>

Best Practices

  • Always declare the document type at the beginning.
  • Ensure that your HTML is well-formed with proper opening and closing tags.
  • Use semantic elements (like <header>, <nav>, <main>, <footer>) to improve accessibility and SEO.
  • Include the alt attribute in images to aid users who rely on screen readers.

This tutorial serves as a starting point for learning HTML. As you become more familiar with these basics, explore additional topics such as CSS for styling and JavaScript for interactivity.
